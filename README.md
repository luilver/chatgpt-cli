# chatgpt-cli
A comand line interface to chatGPT

### Setup
Clone this repo

Install chatgpt ruby gem:
```bash
gem install chatgpt-ruby
```

Copy your api key

Run:
```
export CHATGPT_API_KEY=«PASTE_YOUR_API_KEY_HERE_WITHOUT_QUOTES»
ruby cli.rb
```

### Example
```bash
export CHATGPT_API_KEY=sk-foo123bar456baz789; ruby cli.rb
>>
```

Voilà
